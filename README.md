# 🍲 My Recipe Box - Full Stack Application

A robust, full-stack recipe sharing platform built as a capstone internship project. It allows users to securely register, publish complex structured recipes, interact via comments and ratings, and discover new dishes using a custom search engine.

## 🚀 Tech Stack
* **Frontend:** React.js, Context API, CSS3 (Responsive Grid)
* **Backend:** Node.js, Express.js, RESTful API Design
* **Database:** MongoDB (Mongoose ORM)
* **Security:** JSON Web Tokens (JWT), Bcrypt Password Hashing

## ✨ Key Features
* **Advanced Content Creation:** Users can publish recipes with dynamic ingredient blueprints and step-by-step instructions.
* **Discovery Engine:** Real-time text filtering and tag-based search indexing.
* **Social Interaction:** Integrated 5-star rating system and comment threads on individual recipe cards.
* **Authentication:** Secure user registration and login flows.

## 💻 Local Setup Instructions
1. Clone the repository.
2. Navigate to `/recipebox-backend` and run `npm install`, then `node server.js` to start the database connection.
3. Open a second terminal, navigate to `/recipebox-frontend`, run `npm install`, then `npm start` to launch the React UI.
4. Open `http://localhost:3000` in your browser.
